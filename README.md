# DevFest2019
Apresentacao de prototipo de IoT controle de lampada no DevFest2019 em Maputo. 
<p>Vamos conectar directamente o nosso dispositivo (ESP8266-01) na nuvem do Firebase.</p>


## RECURSOS USADOS
- ESP8266 ESP-01
- Arduino UNO
- Módulo Relé WiFi 
- Smartphone
- Computador
- 1 Extensão de tomadas
- 1 Lâmpada
- Fios
